# 通用 FMEA 模板说明

> 兜底模板，适用于未明确分类的产品/工艺的 DFMEA 与 PFMEA 分析。当其他模板无法明确匹配时使用本模板。

## 适用场景

- 用户未明确产品类型
- 产品不属于电子电气/机械装配/表面处理/涂装任一类
- 多类型混合产品（如机电一体化产品）
- 用户希望从零开始构建 FMEA

## 通用产品类型

- 紧固件（通用）
- 橡胶件（密封圈/减震垫）
- 塑料件（注塑/挤出）
- 玻璃件
- 铸造件
- 烧结件
- 复合材料件
- 机电一体化产品

## 通用失效模式

### 功能类失效

- 功能丧失（无法工作）
- 功能退化（性能下降）
- 功能间歇（偶发故障）
- 部分功能丧失
- 非预期功能
- 功能超范围
- 功能延迟

### 装配类失效

- 错装
- 漏装
- 反装
- 间隙超差
- 预紧力不足

### 尺寸/外观类失效

- 关键尺寸超差
- 形位公差超差
- 表面质量缺陷
- 外观色差

## 失效起因来源（4M1E/6M 通用）

| 6M | 失效起因示例 |
|---|---|
| 人（Man） | 操作失误、培训不足、疲劳作业、未按 SOP |
| 机（Machine） | 设备故障、磨损、校准失效、参数漂移 |
| 料（Material） | 来料超差、批次差异、材料变更、运输损伤 |
| 法（Method） | SOP 缺失、参数错误、监控不足、防错失效 |
| 环（Environment） | 温湿度异常、洁净度不足、振动、电磁干扰 |
| 测（Measurement） | 量具误差、测量方法错误、测量系统失效 |

## S/O/D 评分通用原则

参考 `references/sod_scoring_tables.md` 的标准评分表。

## 特殊特性识别

- **CC**：S=9-10 的失效影响
- **SC**：S=8 且 AP=H/M 的失效影响
- **OSS**：影响操作员安全的特性

## 优化措施建议

### 预防控制（PC）

- 优化设计规范
- 增加 DOE 试验设计
- 增加仿真分析
- 增加可靠性试验
- 加强供应商管理
- 增加防错设计

### 探测控制（DC）

- 增加测试项目
- 提前测试时间
- 采用更敏感的测试方法
- 增加 100% 在线检测
- 增加可靠性测试

## 使用本模板的建议

1. **从结构分析开始**：先构建结构树/过程流程图，明确分析对象
2. **从功能分析入手**：明确每个要素/工序的功能，再推导失效
3. **借鉴其他模板**：参考 electronic-ecm / mechanical-assembly 等模板的失效链案例
4. **结合 references**：参考 `failure_chain_examples.md` 的通用案例
5. **特殊特性识别**：识别出的 CC/SC 应同步到控制计划

## 与其他模板的差异

| 维度 | generic-fmea | 其他行业模板 |
|---|---|---|
| 适用范围 | 兜底，所有产品 | 特定行业 |
| 失效链预填 | 通用，需用户补充 | 行业特定，预填具体案例 |
| S/O/D 提示 | 通用 | 行业典型场景 |
| 特殊特性 | 通用 | 行业典型特性 |
