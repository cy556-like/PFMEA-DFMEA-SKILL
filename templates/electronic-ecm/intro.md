# 电子电气 FMEA 模板说明

> 适用于 ECU、传感器、线束、PCBA、连接器、继电器等电子电气产品的 DFMEA 与 PFMEA 分析。

## 适用产品

- ECU（电子控制单元）
- 传感器（温度/压力/位置/加速度）
- 线束总成
- PCBA（印制电路板组件）
- 连接器
- 继电器
- 电机驱动器
- LED 模组
- 雷达/摄像头模组

## 典型失效模式

### 电气类失效

- 短路（电源对地、信号对地、相间短路）
- 开路（导线断、焊点开裂、连接器松动）
- 过流（IC 损坏、过载）
- 过压（浪涌、ESD 损坏）
- 欠压（电源跌落、电池欠压）

### 功能类失效

- 功能丧失（完全不工作）
- 功能退化（性能下降）
- 功能间歇（偶发故障）
- 非预期功能（误触发、错误响应）
- 功能延迟（响应时间超标）

### EMC 类失效

- 电磁辐射超标
- 电磁敏感度不足
- 信号串扰
- 电磁干扰导致功能异常

## 失效起因来源

### 设计缺陷

- 元件选型不当（耐压/耐流/耐温不足）
- 电路设计缺陷（保护电路缺失）
- PCB 布局不合理（EMI 辐射大）
- 热设计不足（散热面积不够）
- 容差设计不当（公差累积超差）

### P-图噪声因素

- 组件间变化：元件参数离散
- 随时间变化：元件老化、焊点开裂
- 顾客使用：频繁开关、超长时间工作
- 外部环境：高温、高湿、振动、盐雾
- 系统交互：EMC 干扰、地线干扰

### 制造工艺缺陷（PFMEA）

- 焊接缺陷（虚焊、桥连、错件）
- 装配缺陷（错装、漏装、反装）
- 测试覆盖不足（漏测、误判）

## S 评分典型场景

| S | 场景 |
|---|---|
| 10 | 影响行车安全（如制动 ECU 失效） |
| 9 | 不符合法规（如 EMC 超标） |
| 8 | 失去主要功能（如发动机 ECU 无法工作） |
| 7 | 降低主要功能（如传感器精度下降） |
| 6 | 失去次要功能（如指示灯不亮） |
| 4 | 外观问题（如壳体划伤） |

## O 评分典型场景

| O | 场景 |
|---|---|
| 10 | 全新电池化学体系首次量产 |
| 9 | 公司首次使用 SiC 功率器件 |
| 7 | 新型号 IC 首次应用 |
| 5-6 | 成熟 IC + 类似应用 |
| 3-4 | 已验证的成熟设计 |
| 1-2 | 长期量产无失效 |

## D 评分典型场景

| D | 场景 |
|---|---|
| 10 | 无任何测试计划 |
| 9 | 通用功能测试，未针对特定失效 |
| 7 | 测试方法新开发，时间较迟 |
| 5-6 | 已验证测试，通过/不通过 |
| 3-4 | 已验证测试，失效测试，时间充分 |
| 1-2 | 老化测试 + 100% 在线检测 |

## 特殊特性识别

- **CC（关键特性）**：S=9-10 的失效影响
  - 例：制动 ECU 电源短路 → S=10 → CC
- **SC（特殊特性）**：S=8 且 AP=H/M
  - 例：传感器信号漂移 → S=7 + AP=H → SC

## 优化措施建议

### 预防控制（PC）改进

- 增加仿真分析（热仿真、EMC 仿真、应力仿真）
- 增加可靠性试验（HALT、HASS）
- 增加元件筛选（100% 老化筛选）
- 优化电路设计（增加保护电路、冗余设计）
- 加强供应商管理（来料 SPC 监控）

### 探测控制（DC）改进

- 增加测试项目（功能测试 + 性能测试 + 可靠性测试）
- 提前测试时间（设计阶段 → 试产阶段 → 量产阶段）
- 采用更敏感的测试方法（失效测试代替通过/不通过）
- 增加 100% 在线自动检测
- 增加老化测试覆盖率

## 与其他模板的差异

| 维度 | electronic-ecm | mechanical-assembly |
|---|---|---|
| 主要失效模式 | 电气失效、EMC | 机械断裂、磨损 |
| 失效起因 | 设计缺陷 + 元件老化 | 材料缺陷 + 工艺缺陷 |
| 测试方法 | 电气测试、EMC 测试 | 拉伸测试、硬度测试 |
| 特殊特性 | CC 居多（涉及安全/法规） | CC 较少，SC 居多 |
