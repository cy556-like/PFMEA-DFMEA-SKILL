# 机械装配 FMEA 模板说明

> 适用于齿轮、轴承、紧固件、轴、壳体、支架等机械产品的 DFMEA 与 PFMEA 分析。

## 适用产品

- 齿轮（直齿轮/斜齿轮/锥齿轮/行星齿轮）
- 轴承（球轴承/滚子轴承/滑动轴承）
- 紧固件（螺栓/螺母/铆钉）
- 轴类零件（传动轴/曲轴/凸轮轴）
- 壳体（变速箱壳体/电机壳体）
- 支架（结构件/安装支架）
- 装配总成（变速器/减速器/差速器）

## 典型失效模式

### 机械类失效

- 断裂（瞬时断裂/疲劳断裂）
- 磨损（粘着磨损/磨粒磨损/腐蚀磨损）
- 塑性变形（过载变形）
- 松动（紧固件松动/压配合松动）
- 卡滞（运动件卡死）

### 装配类失效

- 错装（错件/反装/漏装）
- 间隙超差（间隙过大/过小）
- 预紧力不足（紧固件扭矩不足）
- 同轴度超差

### 尺寸类失效

- 关键尺寸超差
- 形位公差超差（同轴度/垂直度/平面度）
- 表面粗糙度超差

## 失效起因来源

### 设计缺陷

- 材料选型不当（强度/硬度/韧性不足）
- 几何设计缺陷（应力集中/截面突变）
- 容差设计不当（公差累积超差）
- 热处理设计不当（硬度梯度不合理）

### 制造工艺缺陷（PFMEA）

- 加工工艺缺陷（尺寸超差/形位公差超差/表面粗糙度超差）
- 热处理工艺缺陷（硬度不足/变形/裂纹）
- 装配工艺缺陷（错装/漏装/扭矩不足）
- 焊接工艺缺陷（虚焊/焊穿/焊渣）

### 4M1E 分类

| 4M1E | 失效起因示例 |
|---|---|
| 人（Man） | 操作失误、培训不足、疲劳作业 |
| 机（Machine） | 设备精度下降、刀具磨损、夹具松动 |
| 料（Material） | 来料超差、材料批次差异、材料缺陷 |
| 法（Method） | SOP 缺失、参数错误、防错失效 |
| 环（Environment） | 温湿度异常、振动、洁净度不足 |
| 测（Measurement） | 量具误差、测量方法错误 |

## S 评分典型场景

| S | 场景 |
|---|---|
| 10 | 失去动力传输（如齿轮断裂导致车辆失控） |
| 9 | 不符合法规（如制动性能不达标） |
| 8 | 失去主要功能（如变速器无法换挡） |
| 7 | 降低主要功能（如变速器异响） |
| 6 | 失去次要功能（如指示功能失效） |
| 4 | 外观问题（如壳体划伤） |

## O 评分典型场景

| O | 场景 |
|---|---|
| 10 | 全新材料首次量产 |
| 9 | 公司首次应用新工艺 |
| 7 | 新型号材料首次应用 |
| 5-6 | 成熟工艺 + 类似应用 |
| 3-4 | 已验证的成熟设计 |
| 1-2 | 长期量产无失效 |

## D 评分典型场景

| D | 场景 |
|---|---|
| 10 | 无任何测试计划 |
| 9 | 通用检验，未针对特定失效 |
| 7 | 检验方法新开发，时间较迟 |
| 5-6 | 已验证检验，通过/不通过 |
| 3-4 | 已验证检验，失效测试，时间充分 |
| 1-2 | 100% 在线自动检测 + 老化测试 |

## 特殊特性识别

- **CC**：S=9-10 的失效影响
  - 例：传动轴断裂 → S=10 → CC
- **SC**：S=8 且 AP=H/M
  - 例：轴承早期失效 → S=8 + AP=H → SC

## 优化措施建议

### 预防控制（PC）改进

- 增加仿真分析（FEA 应力分析、运动学仿真）
- 增加可靠性试验（疲劳试验、磨损试验）
- 增加材料筛选（100% 探伤、硬度筛选）
- 优化设计（增加圆角、优化截面、减少应力集中）
- 加强供应商管理（来料 SPC 监控）

### 探测控制（DC）改进

- 增加测试项目（尺寸测量 + 形位公差 + 表面质量）
- 提前测试时间（首件检验 + 过程检验）
- 采用更敏感的测试方法（三坐标代替卡尺）
- 增加 100% 在线自动检测
- 增加可靠性测试覆盖率

## 与其他模板的差异

| 维度 | mechanical-assembly | electronic-ecm |
|---|---|---|
| 主要失效模式 | 机械断裂、磨损 | 电气失效、EMC |
| 失效起因 | 材料缺陷 + 工艺缺陷 | 设计缺陷 + 元件老化 |
| 测试方法 | 拉伸测试、硬度测试 | 电气测试、EMC 测试 |
| 特殊特性 | CC 较少，SC 居多 | CC 居多（涉及安全/法规） |
