# 表面处理 FMEA 模板说明

> 适用于电镀、热处理、氧化、淬火、渗碳等表面处理工艺的 PFMEA 分析。

## 适用工艺

- 电镀（镀锌/镀镍/镀铬/镀铜）
- 热处理（淬火/回火/正火/退火）
- 化学热处理（渗碳/渗氮/碳氮共渗）
- 表面氧化（阳极氧化/发黑/磷化）
- 表面强化（喷丸/喷砂/滚压）
- 涂层（PVD/CVD/热喷涂）

## 适用产品

- 金属支架（镀锌件）
- 齿轮（渗碳淬火件）
- 铝合金外壳（阳极氧化件）
- 弹簧（喷丸强化件）
- 螺栓（发黑/磷化件）
- 工模具（PVD 涂层件）

## 典型失效模式

### 镀层/涂层类失效

- 镀层附着力不足
- 镀层厚度不足
- 镀层厚度不均
- 镀层起泡/起皮
- 镀层色差

### 热处理类失效

- 硬度不足
- 硬度不均
- 变形超差
- 裂纹
- 脱碳
- 过烧

### 表面质量类失效

- 表面粗糙度超差
- 表面污染
- 表面缺陷（裂纹/夹杂）

## 失效起因来源（4M1E）

| 4M1E | 失效起因示例 |
|---|---|
| 人（Man） | 操作未按 SOP、培训不足、未及时更换槽液 |
| 机（Machine） | 设备温度控制失灵、电源不稳、槽液循环不足 |
| 料（Material） | 来料表面油污、槽液成分偏离、添加剂失效 |
| 法（Method） | 工艺参数不当（温度/时间/电流密度）、SOP 缺失 |
| 环（Environment） | 槽液温度波动、车间湿度异常、洁净度不足 |
| 测（Measurement） | 测厚仪校准失效、硬度计误差、附着力测试方法不当 |

## S 评分典型场景

| S | 场景 |
|---|---|
| 10 | 热处理裂纹导致断裂（影响安全） |
| 9 | 不符合法规（如 RoHS 不达标） |
| 8 | 失去主要功能（如镀层脱落导致腐蚀失效） |
| 7 | 降低主要功能（如硬度不足导致早期磨损） |
| 6 | 失去次要功能（如外观色差） |
| 4 | 外观问题（如镀层颗粒） |

## O 评分典型场景

| O | 场景 |
|---|---|
| 10 | 全新工艺首次量产 |
| 9 | 公司首次应用新槽液 |
| 7 | 新工艺参数首次应用 |
| 5-6 | 成熟工艺 + 类似产品 |
| 3-4 | 已验证的成熟工艺 |
| 1-2 | 长期量产无失效 |

## D 评分典型场景

| D | 场景 |
|---|---|
| 10 | 无任何检验 |
| 9 | 通用外观检验，未针对特定失效 |
| 7 | 检验方法新开发，时间较迟 |
| 5-6 | 已验证检验，通过/不通过 |
| 3-4 | 已验证检验，失效测试，时间充分 |
| 1-2 | 100% 在线自动检测 |

## 特殊特性识别

- **CC**：S=9-10 的失效影响
  - 例：齿轮热处理裂纹 → S=10 → CC
- **SC**：S=8 且 AP=H/M
  - 例：镀层附着力不足导致腐蚀 → S=8 + AP=H → SC

## 优化措施建议

### 预防控制（PC）改进

- 优化工艺参数（DOE 试验设计）
- 增加槽液自动监测（pH/浓度/温度）
- 加强供应商管理（来料清洁度）
- 增加工艺验证（CPK 监控）
- 增加设备校准频次

### 探测控制（DC）改进

- 增加测试项目（附着力 + 厚度 + 耐腐蚀）
- 增加 100% 在线检测（如 X-Ray 测厚）
- 增加可靠性测试（盐雾试验 + 老化试验）
- 增加首件检验 + 过程检验频次

## 与其他模板的差异

| 维度 | surface-treatment | painting-coating |
|---|---|---|
| 工艺对象 | 金属表面改性 | 表面涂装 |
| 主要失效 | 镀层/硬度/变形 | 漆面颗粒/流挂/色差 |
| 测试方法 | 附着力/硬度/盐雾 | 目视/色差仪/光泽度 |
| 特殊特性 | CC 较多（涉及功能） | CC 较少（多为外观） |
