# 涂装 FMEA 模板说明

> 适用于喷涂、电泳、漆面、喷漆等涂装工艺的 PFMEA 分析。

## 适用工艺

- 喷漆（空气喷漆/无气喷漆/静电喷涂）
- 电泳涂装（阳极电泳/阴极电泳）
- 粉末喷涂
- 喷塑
- 烤漆
- 修补涂装

## 适用产品

- 保险杠涂装件
- 车身喷涂件
- 车门喷涂件
- 内饰涂装件
- 铝合金轮辋涂装件
- 电机外壳喷涂件
- 金属支架喷涂件

## 典型失效模式

### 漆膜外观类失效

- 漆面颗粒
- 流挂
- 橘皮
- 色差
- 光泽度不足
- 咬底
- 缩孔
- 针孔

### 漆膜性能类失效

- 附着力不足
- 漆膜厚度不足
- 漆膜厚度不均
- 硬度不足
- 耐腐蚀性不足
- 耐候性不足

### 工艺过程类失效

- 漏喷
- 错喷（颜色错误）
- 喷涂位置错误
- 喷涂顺序错误

## 失效起因来源（4M1E）

| 4M1E | 失效起因示例 |
|---|---|
| 人（Man） | 操作工调漆比例错误、喷涂手法不当、未识别缺陷 |
| 机（Machine） | 喷涂设备参数偏离、机器人轨迹错误、喷枪堵塞 |
| 料（Material） | 油漆批次差异、稀释剂比例错误、油漆过期 |
| 法（Method） | 喷涂 SOP 不当、参数设置错误、晾干时间不足 |
| 环（Environment） | 喷漆室洁净度不足、温湿度异常、空气颗粒物污染 |
| 测（Measurement） | 色差仪校准失效、膜厚仪误差、目视检验标准不清 |

## S 评分典型场景

| S | 场景 |
|---|---|
| 9 | 不符合法规（如 VOC 超标） |
| 8 | 失去主要功能（如漆膜大面积脱落导致基体腐蚀） |
| 7 | 降低主要功能（如耐腐蚀性下降） |
| 6 | 失去次要功能（如局部漆膜缺陷） |
| 5 | 降低次要功能（如光泽度下降） |
| 4 | 外观非常不舒服（如漆面颗粒密集） |
| 3 | 外观中度不舒服（如局部色差） |
| 2 | 外观略微不舒服（如轻微橘皮） |

## O 评分典型场景

| O | 场景 |
|---|---|
| 10 | 全新涂装体系首次量产 |
| 9 | 公司首次应用新涂料 |
| 7 | 新颜色首次量产 |
| 5-6 | 成熟工艺 + 类似产品 |
| 3-4 | 已验证的成熟工艺 |
| 1-2 | 长期量产无失效 |

## D 评分典型场景

| D | 场景 |
|---|---|
| 10 | 无任何检验 |
| 9 | 通用目视检验，未针对特定失效 |
| 7 | 检验方法新开发，时间较迟 |
| 5-6 | 已验证目视检验 + 抽样测厚 |
| 3-4 | 100% 目视 + 色差仪 + 测厚仪 |
| 1-2 | 100% 在线自动检测 + AI 视觉 |

## 特殊特性识别

- **CC**：S=9-10 的失效影响（涂装行业较少）
- **SC**：S=7-8 且 AP=H/M
  - 例：漆膜附着力不足导致大面积脱落 → S=8 + AP=H → SC

## 优化措施建议

### 预防控制（PC）改进

- 升级喷漆室洁净度（高效过滤）
- 增加温湿度自动控制
- 增加调漆自动计量
- 增加机器人轨迹验证
- 加强供应商管理（油漆 SPC）
- 增加油漆批次管理

### 探测控制（DC）改进

- 增加测试项目（外观 + 膜厚 + 附着力 + 耐腐蚀 + 耐候）
- 增加 100% 在线检测（AI 视觉）
- 增加可靠性测试（盐雾 + 老化）
- 增加首件检验 + 过程检验频次

## 与其他模板的差异

| 维度 | painting-coating | surface-treatment |
|---|---|---|
| 工艺对象 | 表面涂装（有机涂层） | 表面改性（无机层/金属层） |
| 主要失效 | 漆面颗粒/流挂/色差 | 镀层/硬度/变形 |
| 测试方法 | 目视/色差仪/光泽度 | 附着力/硬度/盐雾 |
| 特殊特性 | CC 较少（多为外观） | CC 较多（涉及功能） |
