# 失效链（FE→FM→FC）案例库

> 本文件汇总各行业的典型失效链案例，供 FMEA 分析时参考。每个案例包含完整的失效影响（FE）、失效模式（FM）、失效起因（FC）及 S/O/D 评分与 AP 判定。

---

## 一、失效链基本概念

```
失效影响（FE）── 失效模式（FM）── 失效起因（FC）
   ↑                ↑                ↑
为什么？         发生了什么？      为什么？
（后果）         （现象）         （根因）
```

- **FE（Failure Effect）**：失效对上级要素/最终用户/法规的影响
- **FM（Failure Mode）**：关注要素的功能失效方式
- **FC（Failure Cause）**：下级要素失效或噪声因素导致 FM

## 二、电子电气行业案例（适用模板：electronic-ecm）

### 案例 2.1：ECU 电源管理模块过热失效

| 字段 | 内容 |
|---|---|
| **产品** | ECU 控制单元 |
| **失效链 ID** | ECU-001 |
| **FE（失效影响）** | ECU 在客户使用中过热失效，导致发动机控制异常，可能引起车辆行驶中熄火，影响行车安全 |
| **FM（失效模式）** | 电源管理 IC 热失控 |
| **FC（失效起因）** | IC 散热设计不足，热阻超标导致结温超过最高允许温度（设计缺陷——散热面积不足 + P-图噪声因素：外部环境高温） |
| **S（严重度）** | 10（影响行车安全，可能危及驾驶员） |
| **O（频度）** | 4（成熟 IC + 类似应用，预防控制部分有效） |
| **D（探测度）** | 6（已验证高温老化测试，但仅通过/不通过判定） |
| **AP（行动优先级）** | **H**（S=10 几乎必为 H） |
| **PC（预防控制）** | 热仿真分析 + IC 选型规范 + 散热基板设计规范 + 设计评审 |
| **DC（探测控制）** | 高温老化测试 1000h + 红外热成像 + 客户现场温度监测 |
| **建议措施** | 1. 增加散热铜箔面积 50%<br>2. 改用更低热阻的封装<br>3. 增加温度保护电路 |
| **特殊特性** | CC（S=10 关键特性） |

### 案例 2.2：传感器信号漂移

| 字段 | 内容 |
|---|---|
| **产品** | 温度传感器 |
| **失效链 ID** | SEN-001 |
| **FE（失效影响）** | 温度测量值漂移，导致 ECU 控制策略错误，可能引起发动机性能下降 |
| **FM（失效模式）** | 传感器输出信号缓慢漂移（功能退化） |
| **FC（失效起因）** | 传感器内部电阻随时间老化（P-图噪声因素：随时间变化） |
| **S（严重度）** | 7（预期寿命内降低主要功能） |
| **O（频度）** | 5（成熟技术，类似应用，预防控制部分有效） |
| **D（探测度）** | 7（已验证测试方法，但测试时间较迟，量产阶段才发现） |
| **AP（行动优先级）** | **H**（S=7 + O=5 + D=7） |
| **PC（预防控制）** | 元件选型规范 + 老化筛选规范 + 供应商管理 |
| **DC（探测控制）** | 量产前高温老化测试 + 客户现场定期校准 |
| **建议措施** | 1. 供应商出厂前 100% 老化筛选<br>2. ECU 软件增加温度漂移自诊断<br>3. 缩短客户现场校准周期 |
| **特殊特性** | SC（S=7 + AP=H 特殊特性） |

| **建议措施** | 1. 增加 Poka-Yoke 防错<br>2. 增加物料条码扫描<br>3. 增加视觉防错系统 |
| **特殊特性** | 无（S=6 + AP=M，不识别为 SC） |

### 案例 6.2：尺寸超差

| 字段 | 内容 |
|---|---|
| **产品** | 通用机加件 |
| **失效链 ID** | DIM-001 |
| **FE（失效影响）** | 零件关键尺寸超差，导致装配困难或功能下降 |
| **FM（失效模式）** | 关键尺寸超差（功能超范围） |
| **FC（失效起因）** | 设备磨损 + 刀具寿命管理不足（机 + 法） |
| **S（严重度）** | 5（降低次要功能） |
| **O（频度）** | 5（成熟工艺，类似应用） |
| **D（探测度）** | 4（已验证 SPC 监控，时间充分） |
| **AP（行动优先级）** | **M**（S=5 + O=4-5 → M） |
| **PC（预防控制）** | 刀具寿命管理规范 + 设备点检 + SPC 监控 |
| **DC（探测控制）** | 首件检验 + SPC 抽检 + 三坐标测量 |
| **建议措施** | 1. 增加刀具寿命预警<br>2. 缩短设备点检周期<br>3. 增加 SPC 控制项 |
| **特殊特性** | 无 |

### 案例 6.3：来料不良

## 七、案例库使用指南

### 7.1 如何使用案例库

1. **根据产品类型选择参考案例**：
   - 电子电气产品 → 第 2 章案例
   - 机械装配产品 → 第 3 章案例
   - 表面处理产品 → 第 4 章案例
   - 涂装产品 → 第 5 章案例
   - 其他 → 第 6 章通用案例

2. **从案例中提取模式**：
   - 识别失效模式类型（功能丧失/退化/间歇等）
   - 识别失效起因来源（4M1E 分类）
   - 评估 S/O/D 评分（参考案例的评分思路）

3. **不可直接复制案例**：
   - 案例仅供参考，应根据实际产品调整
   - 案例的 S/O/D 评分基于特定假设，实际评分需基于真实情况

- 后续版本将增加更多行业案例（如新能源电池、智能驾驶等）
